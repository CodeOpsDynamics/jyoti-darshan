### 7.3 Debugging Process

**Bug Example: Chatbot Same Response Issue**

**Step 1: I Discovered Bug** (through testing)
- Tested with "Will I find love?" → Got generic response
- Tested with "What about career?" → Got same generic response
- **Identified problem: keyword detection not working**

**Step 2: I Diagnosed** (analyzed code)
```javascript
// Original broken code
function generateAIResponse(question) {
  if (question.includes('love')) { ... }
}
```
**Problem I identified:** Case-sensitive matching (user typed "Love" not "love")

**Step 3: I Asked AI for Fix**
> "My chatbot keyword detection is case-sensitive. Fix it to work with any capitalization."

**Step 4: AI Provided Solution**
```javascript
function generateAIResponse(question) {
  const lowerQ = question.toLowerCase();
  if (lowerQ.includes('love')) { ... }
}
```

**Step 5: I Tested Fix**
- Tested 20+ questions with various capitalizations
- Verified all 10 categories working
- **Validated fix successful**

**Key Point:** I found bug, diagnosed issue, specified fix needed. AI implemented MY solution.
